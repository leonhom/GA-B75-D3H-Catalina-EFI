# GA-B75-D3H-Catalina-EFI
EFI folder for GA-B75-D3H/E3 1230v2/RX480 8G Hackintosh(macOS 10.15.1 Catalina)

## 更新 2020/01/30

- 2020/01/30：已正常使用使用2个月，无异常情况。

## 配置清单
- CPU：Intel Xeon E3-1230 V2
- 主板：技嘉B75D3H
- 显卡：蓝宝石RX480 8G
- SSD：三星860Evo-256G
- 内存：DDR3 1600 8G x 4
- 网卡：博通BCM943602CS - 4天线
